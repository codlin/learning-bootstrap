Bootstrap5 表单

Bootstrap5 提供了三种不同类型的表单布局：

  • 垂直表单（默认）
  • 水平表单
  • 内联表单

创建垂直表单

基本的表单结构是Bootstrap自带的，个别的表单控件自动接收一些全局样式。

把标签和控件放在一个带有 .form-group 类的 div 元素中，这是获取最佳间距所必须的。

label 元素添加了 .form-label 类以确保正确的填充。

向所有的文本输入元素 input、texttarea 添加 .form-control 类。

创建内联表单

表单元素并排显示，可以使用 .row 和 .col

表单控件尺寸

.form-control-lg 或 .form-control-sm 更改 .form-control 输入控件的大小。

向表单控件添加帮助文本

必须8-20个字符长度。

表单下拉列表

将 .form-select 类添加到 select 元素。

使用 .form-select-lg 或 .form-select-sm 改变下拉列表的大小。

添加 multiple 属性使下拉列表多选

复选框和单选框

复选框有不同的标记，它们被设置了 .form-check 类的容器元素包围。 label 设置 .form-check-label 类，而复选框和单选按钮使用 .form-check-input。 希望默认选中复选框，可以使用 checked 属性。

复选框

使用 .form-check 包装元素，来确保标签和复选框有合适的外边距。

单选框

拨动开关

将 .form-check 和 .form-switch 类一起在容器中使用，可以将复选框设置为开关样式。

滑块

.form-range 类添加到类型为 range 的 input 元素，可以设置范围菜单的样式。

表单组

.input-group 类是一种容器，通过在输入字段的前面或后面添加图标、文本或作为“帮助文本“来增强输入。

.input-group-text 类指定帮助文本的样式。
